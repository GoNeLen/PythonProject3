# Personal Finance Manager

Desktop приложение для учета личных финансов.

## Функциональность
- Учет доходов и расходов
- Категоризация операций
- Расчет текущего баланса
- Аналитика по категориям
- Автосохранение в CSV-файл

## Запуск
1. Установите Python 3.7+
2. Запустите: `python finance_gui.py`
3. Добавляйте операции через форму вверху
4. Просматривайте аналитику через кнопку "Аналитика"
Отличия от предыдущей задачи:
Новая тематика: финансы вместо задач

Другая структура данных: CSV вместо JSON

Более сложная бизнес-логика: расчет баланса, аналитика

Расширенный GUI: таблица, комбобокс, аналитика в отдельном окне

Использование Enum для типов операций
